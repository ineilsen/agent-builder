# UK Public Sector AI Marketplace

A comprehensive replica of the UK Public Sector AI Marketplace built with React, Vite, and Tailwind CSS.

## Features

- **Interactive Navigation**: 5 main tabs (Home, MHCLG Apps, Responsible AI, Agent Builder, MCP Marketplace)
- **Hero Section**: Eye-catching hero with trust badges and compliance metrics
- **Statistics Dashboard**: 4 key metrics cards showing platform performance
- **Department Explorer**: 5 government department cards with tailored solutions
- **Featured AI Solutions**: Showcase of 4 AI applications with detailed information
- **How It Works**: 3-step process visualization
- **Call-to-Action**: Prominent CTA section for user engagement
- **Comprehensive Footer**: Multi-column footer with all necessary links

## Tech Stack

- **React 18**: Modern React with hooks
- **Vite**: Fast build tool and dev server
- **Tailwind CSS v4**: Utility-first CSS framework
- **React Router v6**: Client-side routing
- **Lucide React**: Beautiful icon set

## Getting Started

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

The application will be available at `http://localhost:5173/`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
